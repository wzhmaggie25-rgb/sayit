# Typeless 颗粒度逆向手册

## 一、计划
1. 解包ASAR: npx @electron/asar extract
2. 文件扫描分类: HTML→模块→JS→CSS
3. 优先级: HTML > .mjs > 小.js > 主进程 > 大bundle

## 二、实施
1. HTML页面: 提取<script>/<link>映射
2. 字符串表: 正则提取所有字面量
3. IPC通道: 正则[\w:-]{10,60}匹配
4. 状态机: 搜索xstate/event关键词
5. 窗口配置: 搜索BrowserWindow/width/transparent
6. 类层级: 搜索class X extends Y

## 三、难点解决
1. 混淆代码: 通过上下文推断语义
2. 自动拦截: 添加Bash权限规则
3. 大文件: Python re.finditer流式搜索
4. GBK编码: utf-8 errors=replace
5. 字符串加密: 提取解密函数+数组

## 四、经验
1. 字符串提取最有效(4553个→功能全景)
2. IPC提取次之(182个→通信架构)
3. HTML映射第3(6页面→组件路由)
4. AudioWorklet未被压缩(意外发现)
5. SendMessage是Typeless注入方式
