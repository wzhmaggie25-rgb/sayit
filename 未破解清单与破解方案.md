## 开源破解方案

### 方案A: webcrack (推荐, GitHub 2.5k stars)
```bash
npm install -g webcrack
webcrack dist/main/index.js -o cracked/
```
https://github.com/j4k0xb/webcrack
- 专攻 webpack bundle 反混淆
- 恢复原始模块结构
- 自动 beautify

### 方案B: repkg
```bash
npx repkg unpack dist/main/index.js
```
https://github.com/nicepkg/repkg

### 方案C: de4js
https://lelinhtinh.github.io/de4js/

```python
import jsbeautifier
opts = jsbeautifier.default_options()
opts.indent_size = 2
beautiful = jsbeautifier.beautify(obfuscated_code)
```
