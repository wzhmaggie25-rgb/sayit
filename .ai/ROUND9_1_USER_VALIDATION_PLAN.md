## 目标

只验证 Round 9 / 9.1 修复，不增加新功能。

通过标准：

```text
不丢文字
不重复输入
不破坏剪贴板
一次右 Alt 停止
有输入框时不弹大结果卡片
无输入框且完全未注入时才弹结果卡片
结果卡片位于条形悬浮窗上方
不抢用户主动切换后的焦点
AI失败时不永久卡死
后台异常后UI可恢复
```

## 开始前

1. 关闭 SayIt、Agent Bridge 和 ZCode。
2. 在 PowerShell 执行：

```powershell
cd D:\code\sayit_zcode
git pull --ff-only origin feature/silent-learning-stabilization
git status --short
```

3. `git status --short` 必须无输出。
4. 启动开发版 SayIt。
5. 在运行日志中确认 keyboard helper ABI/version 为 4；若仍为 3 或 unknown，立即停止验收，先重建并确认 DLL。

## A. 结果卡片外观与位置

### A1 无输入框

- 点击桌面空白处，确认没有任何输入光标；
- 录制一句 10–20 字的话；
- 按一次右 Alt 停止。

预期：

- 不向其他软件输入文字；
- 弹出约 360px 宽的小卡片；
- 卡片位于条形悬浮窗正上方约 14px；
- 不位于屏幕中心或左上角；
- 可以复制和关闭；
- 未点击复制前不改变剪贴板。

### A2 长文本

- 在无输入框状态录制 100–200 字；
- 检查卡片高度不超过约 260px；
- 长文本在卡片内部滚动，不把卡片撑满屏幕。

### A3 多显示器（有多屏时）

- 分别在主屏和副屏开始录音；
- 卡片必须跟随对应屏幕的条形悬浮窗；
- 不跨屏、不出界。

## B. 有输入框时不得弹大卡片

分别在以下应用执行至少两次：

```text
记事本
Chrome 普通输入框
Obsidian
微信
飞书
VS Code（普通文本/Markdown，不在代码危险位置测试）
```

每次：

- 点击输入框确认光标；
- 说 10–30 字；
- 按一次右 Alt停止。

预期：

- 文字进入当前输入框；
- 大结果卡片不出现；
- 不重复输入；
- 光标和选区外文字不被破坏。

## C. 连续会话污染

1. 在桌面无输入框录一次，让卡片出现并关闭；
2. 回到记事本连续录制 10 次；
3. 每次使用不同短句。

预期：

- 后续 10 次均不重复弹旧卡片；
- 不出现上一轮文字；
- 不出现旧的复制完成状态；
- 不被旧定时器自动关闭当前会话窗口。

## D. 一次右 Alt 停止

分别录制：

```text
10秒
1分钟
5分钟
```

条件允许再测试 15 分钟。

预期：

- 第二次右 Alt按下一次就进入“思考中/处理中”；
- 不需要按第二次；
- Alt不激活应用菜单；
- 不丢失输入光标；
- 不自动开始下一轮录音。

每组重复 3 次。

## E. 用户主动切换窗口

1. 在记事本输入框开始录音；
2. 录音结束后按右 Alt；
3. 在“思考中”主动切换到另一个应用并点击其输入框。

预期：

- SayIt不得在处理结束后抢回记事本；
- 不得把文字注入错误窗口；
- 无法安全确认目标时，使用轻提示或保存历史，不重复尝试。

## F. 剪贴板保护

分别把剪贴板预先设置为：

```text
普通文本
一张图片
一个文件
```

然后在支持输入的目标里录音。

预期：

- 普通文本恢复原值；
- 图片和文件仍可正常粘贴；
- 结果卡片未点击复制时不得覆盖剪贴板。

## G. AI超时/失败降级

使用测试配置或临时无效网络环境触发一次AI失败。

预期：

- “思考中”不会永久停留；
- 使用本地识别/纠错文本继续；
- 只注入一次；
- 显示“AI整理超时/已使用识别结果”类提示；
- 下一次录音仍可正常使用。

## H. 后台恢复

仅使用已有测试开关和调试方式触发一次 backend异常退出，不手动删除用户数据。

预期：

- 条形悬浮窗退出永久“思考中”；
- 显示后台恢复提示；
- 最多自动重启一次；
- 不自动重放上一次录音；
- 不重复注入；
- 恢复后新录音可正常开始。

## 通过后的动作

只有全部P0项目通过，才允许：

1. 创建用户验收报告；
2. 固定稳定commit和tag；
3. 决定是否合并main；
4. 从稳定点创建 `feature/release-foundation`。

任何以下问题都视为P0失败：

```text
丢文字
重复输入
破坏剪贴板
抢回错误窗口
有输入框仍弹大卡片
一次Alt不能停止
后台崩溃后无法继续使用
```
