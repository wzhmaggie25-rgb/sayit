使用指南

欢迎使用 Sayit — 一款 Windows 桌面语音输入工具，按下热键即可语音转文字。

⚡ 快速开始
1. 注册 API 密钥（见下方说明）
2. 在「设置 → 语音识别」填入密钥
3. 按 右 Alt 开始录音，说出要转写的内容
4. 再次按 右 Alt 结束录音并转写 → 文字自动输入
5. 文字自动输入到目标窗口后，直接在输入框中修改错误，Sayit 自动检测并学习纠错规则

🔑 如何获取 API 密钥
· 阿里云 DashScope — 登录阿里云 → 开通灵积服务 → 获取 sk- 密钥
· 火山方舟 — 登录火山引擎 → 开通语音识别 → 获取 ak- 密钥
· DeepSeek — 登录 DeepSeek → API Keys → 创建新密钥
· ONNX 离线引擎 — 完全免费，零配置，在「语音识别」开启即可断网使用

🎤 离线模式
开启 ONNX 本地引擎后，不需联网也能语音转文字。Paraformer-Small（340MB）推荐，模型已内置，开箱即用。

📖 个人词库
添加词汇可提高识别准确率。开启「静默自学」后，在输入框中手动修正错字，系统自动提取纠错规则存入词库，无需手动添加。

🤫 静默自学
转写完成后，Sayit 监控目标窗口 12 秒。若你在输入框中手动修改了文字，系统通过 UIA/MSAA 读取当前文本，与原文比对提取纠错规则。全程本地、零网络开销、零额外成本。仅对标准编辑控件生效（Chrome / VS Code / Office / 记事本等约 85% 场景），不支持自绘控件（微信 / QQ / 钉钉）。

🧠 AI 人格与自定义指令
人格设定基础行为模式，自定义指令在此基础上追加。两者叠加生效，指令优先级更高。

⌨️ 快捷键
按下 右 Alt 开始录音，再次按下结束并转写，文字自动输入。可在录音界面点击快捷键提示修改绑定。

🪟 管理员模式
如需在管理员终端中输入文字，请以管理员身份启动 Sayit。
