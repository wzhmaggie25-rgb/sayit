# Typeless 逆向 — 剩余 15% 解决方案

## 1. 数据库列名 → SQLite 直接读取

Typeless 数据库位置:
```
%APPDATA%/Typeless/ 或 %APPDATA%/now.typeless.desktop/
```

**方法**: 用 Python sqlite3 直接读表结构:
```python
import sqlite3
conn = sqlite3.connect("path/to/typeless.db")
cursor = conn.execute("SELECT name FROM sqlite_master WHERE type='table'")
for row in cursor: 
    print(row[0])
    cols = conn.execute(f"PRAGMA table_info({row[0]})")
    for c in cols: print(f"  {c[1]} {c[2]}")
```

## 2. AI 模型名 → 设置界面截图 + 配置文件

Typeless 的用户配置存储位置:
```
%APPDATA%/Typeless/config.json 或 Local Storage
```

**方法**: 
- 打开 Typeless → 设置 → AI/语音引擎页
- 截图所有模型选项
- 或在 %APPDATA% 搜索 `*.json` 配置文件

## 3. API 端点路径 → HTTP 代理抓包

**方法A (推荐)**: mitmproxy
```bash
pip install mitmproxy
mitmweb --listen-port 8888
# 设置 Typeless 代理为 127.0.0.1:8888
# 触发录音 → 查看 API 请求
```

**方法B**: Fiddler Classic (Windows)
- 安装 Fiddler → 启动 → 自动捕获 HTTP/HTTPS
- 筛选 `api.typeless.com`

## 4. 浮窗 CSS 精确值 → Electron DevTools

**方法**:
```
# 启动 Typeless, 然后运行:
electron --remote-debugging-port=9222
# 浏览器打开 http://localhost:9222
# 选择 floating-bar 窗口 → Elements 面板
# 查看 React 组件的 computed styles
```

**方法B**: 对着 Typeless 浮窗截图, 用取色器测量:
- 圆角半径
- 内边距
- 颜色值
- 阴影参数
- 字体大小

## 5. SendMessage 参数 → API Monitor

**方法**: 下载 API Monitor (apimonitor.com)
- 选择 Typeless.exe
- 钩住 `user32.dll!SendMessageW`
- 触发文本注入
- 记录 wParam, lParam 参数值

## 6. 键盘 DLL 细节 → DLL Export Viewer

Typeless 的键盘 DLL 路径:
```
%APPDATA%/Typeless/resources/app.asar.unpacked/
```

**方法**:
```bash
# 找到 .dll 或 .node 文件
find "%APPDATA%/Typeless" -name "*.dll" -o -name "*.node"
# 用 DLL Export Viewer 查看导出函数
```
